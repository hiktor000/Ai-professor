تعليمات توجيهية لـ Manus AI لإنشاء تطبيق Android (APK) من مستودع GitHub
================================================================================

اسم التطبيق: الأستاذ الذكي - AI Professor (برعاية Ahmed Grairi)
وصف المشروع: تطبيق تعليمي تفاعلي ذكي مبني باستخدام React + Vite + Express Backend (Node.js) وGemini API.

عزيزي Manus AI،
يرجى تحويل مستودع GitHub هذا إلى تطبيق Android بصيغة APK جاهز للتثبيت والاستخدام بسلاسة وبدون أي أخطاء، مع مراعاة الخطوات والتعليمات التالية:

1. طبيعة المشروع والذكاء الاصطناعي:
   - التطبيق يحتوي على واجهة React واجهة خلفية Node.js/Express في ملف server.ts تقوم بإدارة طلبات الذكاء الاصطناعي ومعالجة النصوص والمعادلات الرياضية (LaTeX/KaTeX).
   - لمعالجة الطلبات البرمجية الخاصة بالـ API بشكل صحيح في الـ APK، يرجى تغليف التطبيق كـ Android WebApp / Capacitor / Bubblewrap / WebView أو ضبط رابط السيرفر الخلفي (Backend Endpoint) ليعمل بسلاسة عبر الشبكة.

2. إعدادات الغلاف (WebView / Capacitor / Bubblewrap):
   - اسم التطبيق المنشأ: الأستاذ الذكي
   - Package Name: com.aiprofessor.ahmedgrairi
   - قم بتفعيل اتصالات الانترنت والميكروفون والملفات في ملف AndroidManifest.xml:
     * android.permission.INTERNET
     * android.permission.ACCESS_NETWORK_STATE
     * android.permission.READ_EXTERNAL_STORAGE / READ_MEDIA_IMAGES
   - اسمح بالوصول الكامل لطلب الـ API وتجاوز شروط CORS أو اتصالات HTTPS إن لزم الأمر.

3. الواجهة والتجاوب (UI/UX Optimization):
   - دعم كامل للغة العربية والاتجاه من اليمين إلى اليسار (RTL).
   - تفعيل شاشة العرض الكاملة (Full Screen / Fit Viewport) على الهواتف الذكية.
   - دعم زر الرجوع الخاص بنظام أندرويد (Hardware Back Button) للتنقل داخل التطبيق بدلاً من إغلاقه فوراً.
   - التأكد من ظهور تنسيقات معادلات LaTeX والرموز الرياضية والفيزيائية بشكل واضح وسريع.

4. المخرجات المطلوبة:
   - قم ببناء ملف APK جاهز للتثبيت المباشر على أجهزة الأندرويد (Release or Debug APK).
   - تأكد من اختبار التطبيق والتحقق من عدم وجود شاشة بيضاء أو أعطال شبكة عند بدء التشغيل.
